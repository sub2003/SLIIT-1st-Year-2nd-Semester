🛠️ Step-by-Step IntelliJ Setup

1Create a New Project in IntelliJ IDEA

Open IntelliJ IDEA → File → New → Project

  • Select Maven Archetype from the left panel
  • Archetype: maven-archetype-webapp
  • GroupId: com.eventhorizon
  • ArtifactId: EventHorizon
  • JDK: 17 or higher
  • Click Create
2Replace pom.xml

Delete the generated pom.xml and paste the content from the provided pom.xml file. Then click the Maven reload icon (🔄) that appears in the top right.

3Create the Java Source Folders

In IntelliJ's Project panel, create these folders under src/main/java/:

com/eventhorizon/model
com/eventhorizon/service
com/eventhorizon/servlet
com/eventhorizon/util

Right-click src/main/java → New → Package → type com.eventhorizon.model → repeat for others.

4Add All Java Files

Right-click each package → New → Java Class → paste the corresponding code:

PackageFiles to Create
modelUser.java, Customer.java, Admin.java, Event.java, Booking.java
serviceUserService.java, EventService.java, BookingService.java
servletUserServlet.java, EventServlet.java, BookingServlet.java
utilFileHandler.java
5Set Up the Web Directory

The Maven webapp archetype creates a src/main/webapp/ folder. Rename or use this as your web folder. If you used the folder structure from this project (web/), tell IntelliJ by going to:

File → Project Structure → Modules → Web → Web Resource Directories → set it to your web/ folder.

✅ Easiest approach: Just use src/main/webapp as your web root and place all JSP/CSS/JS files there directly — Maven webapp archetype already sets this up.
6Add JSP & Static Files

Place these files inside src/main/webapp/ (or web/):

  • index.jsp, login.jsp, register.jsp, events.jsp, eventDetail.jsp, myBookings.jsp, profile.jsp
  • WEB-INF/web.xml
  • css/style.css
  • js/main.js
  • admin/dashboard.jsp, admin/events.jsp, admin/addEvent.jsp, admin/editEvent.jsp, admin/bookings.jsp, admin/users.jsp
7Add Data Files

Create a data/ folder at your project root (outside src/) and add:

  • users.txt — paste sample data
  • events.txt — paste sample data
  • bookings.txt — paste sample data
⚠️ The FileHandler.java saves data to System.getProperty("user.home") + "/eventhorizon_data/" (your home directory). This is automatically created on first run. You can change the DATA_DIR path in FileHandler.java if you prefer a different location.
8Configure Tomcat in IntelliJ
  1. Go to Run → Edit Configurations
  2. Click + → Tomcat Server → Local
  3. Under Server tab: set your Tomcat installation path
    (Download Tomcat 10+ from tomcat.apache.org if not installed)
  4. Under Deployment tab: click + → Artifact → select EventHorizon:war exploded
  5. Set Application context to /EventHorizon
  6. Click OK
✅ IntelliJ Ultimate has built-in Tomcat support. If you have Community Edition, use the Smart Tomcat plugin (free, install via Plugins menu).
9Run the Application
  1. Click the green ▶️ Run button
  2. Browser should open at: http://localhost:8080/EventHorizon/
